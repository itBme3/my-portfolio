## Getting Started

To get a local copy up and running follow these simple example steps.

### **Prerequisites:**

    Node version >= 14

### **Installation:**

1. Clone the repo
   ```sh
   git clone https://github.com/itBme3/my-portfolio
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Generate content
   ```sh
   yarn run generate
   ```
4. Serve Locally
      ```sh
      yarn run start
      ```
   Or Development (hot reloading using Vite)
      ```sh
      yarn run dev
      ```
